# Culture - AI駆動型ニュースキュレーションプラットフォーム

**Culture**は、Google Cloud AI Agent Hackathon向けに開発されたAI駆動型ニュースキュレーションプラットフォームです。

ユーザーの興味・関心に基づいてパーソナライズされたニュース体験を提供し、AIエージェントとの対話を通じてトピックを探索できます。

## 主要機能

- **AI駆動のニュースキュレーション**: Google Gemini 2.0 Flashを活用した記事の要約・カテゴリ分類
- **パーソナライズされたレコメンデーション**: ユーザーの評価履歴に基づくコンテンツ提案
- **インタラクティブチャット**: トピック探索のためのAIエージェント対話機能
- **タグベース検索**: カテゴリ別の記事発見機能
- **ユーザー認証**: NextAuth v5 + Rails JWT認証システム
- **記事評価**: Good/Bad評価機能（排他的選択ロジック）
